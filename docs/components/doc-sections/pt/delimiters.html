Delimitadores (Delimiters)

Por padrão, o Easy traz alguns delimitadores, que são: easy delimiter e o common delimiter que também é usado pelo Vue.js, Angular.js, e outros.

Os delimitadores abrem uma espaço no HTML para execução de um trecho de código javascript, isto significa que podemos executar qualquer código javascript puro como variáveis, funções, operadores ternários, etc. Só precisa retornar um valor.

Sempre que um valor for um objecto ele torna-o em JSON.

Easy Delimiter

Para usar este delimitador basta fazer o seguinte: -e- [Content] -

Common Delimiter

Para usar este delimitador basta fazer o seguinte: {{ [Content] }}

HTML Delimiter (Desde v2.2.0+)

Para usar este delimitador basta fazer o seguinte: {html{ [Property] }}
Que é usado para processar trechos html passado por uma propriedade.

Podemos consultar todos os delimitadores disponíveis na propriedade instance.delimiters. Estes valores são inalteráveis por esta propriedade.

Você pode o seu próprio delimitador, como desejas (Falaremos sobre isso mais depois).
