Introdução (Introduction)

O que é Easy.js? (What is Easy.js?)

Easy.js, forma curta de Easy and Asyncronous Javascript, é uma biblioteca javascript para construção de interfaces de usuário, e auxilia no desenvolvimento de aplicações web, proporcionando uma interação síncrona entre interfaces de usuário e dados no Javascript.

Por quê Easy.js? (Why Easy.js?)

Como o nome diz, é uma biblioteca javascript realmente fácil de usar que fornece a você uma maneira simples de interagir com o seu HTML usando propriedades reactivas, dados em escopo no HTML, componentes reutilizáveis escritos em HTML puro, chamadas de API usando apenas atributos, e assim por diante... É muito bom e simples construir uma aplicação de página única (SPA - Single Page Application). Nenhum DOM Virtual é usado, ele usa o DOM Real para lidar com todas as alterações de dados.

Compatibilidade nos Navegadores

Ela foi testada em quase todos os navegadores populares e no IE 11, as versões abaixo não é suportado, porque ela usa conteitos do ECMAScript 2015 (ES6).

Escala de Projectos

Uma aplicação web totalmente desenvolvida com easy.js foi testado apenas com pequenos projetos, projetos maiores ainda não foram testados.

Essa documentação pressupõem que você já esteja familiarizado com HTML, JavaScript e CSS (opcional). Para cada capítulo, você terá um código de exemplo associado para ajudá-lo a entender as coisas passo a passo.
