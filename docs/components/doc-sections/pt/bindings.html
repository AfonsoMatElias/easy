Bindings

One Way

Podemos alcançar o vinculação de dados unilateral (One Way) simplesmente definindo o delimitador no espaço ou atributo que precisamos que o valor esteja.

Se não quisermos fazer deste jeito, espondo o delimitador no atributo do elemento, ainda podemos usar um comando e-atributo="{{ expressão }}", vai criar o atributo depois do elemento ser compilado. É um comando muito versátil, entraremos em detalhes na secção de Comandos (commands).

Two Way

Podemos alcançar a vinculação de dados bidirecional usando o comando e-bind:atributo="propriedadeDoEscopo", por padrão ele vincula ao attributo value, isto significa que se quisermos vincular ao atributo `value` podemos fazer simplesmente deste jeito e-bind="propriedadeDoEscopo".
