Extra

Easy, also provides you some extensions methods to your code, they are small, simple, easily rememberable, and can be very useful... Let's see them: 


    // HTMLElement.node(...) -> executes querySelector method
    var element = document.node('#node-id'); 
    
    // HTMLElement.nodes(...) -> executes querySelectorAll method, 
    // returns an `Array` of the elements not `NodeList`
    var elements = document.nodes('.node'); 
    
    // HTMLElement.listen(...), executes addEventListener method, fortunately
    // it loops the elements when used with element.nodes(...)
    document.node('#form-id').listen('submit', function(evt){
        ...
    });
    
    document.nodes('.menu-item').listen('click', function(evt){
        // You can access the `event element` at `evt.base` 
        ...
    });

    // The `listen extension` returns:
    [{
        el: Element, // The element having the event
        event: String, // The event name
        callback: Function, // The callback of the event
    }]

    // To remove the Event Listener, use the return Array and the `removeEventListener`
    document.removeEventListener(listenReturn[0].name, listenReturn[0].callback);

    // Note: If you use the callback passed in the listen method won't work!

    // HTMLElement.valueIn(...) -> allow to get and set value into an attribute.

    var value = element.valueIn('id'); // Returns the value of id `property`
    value = element.valueIn('id', 'id-001'); // Sets the attribute id and return the value
    value = element.valueIn(); // If used this way it gets the textContent value

    // HTMLElement.aboveMe(...) -> Get the parent element
    var parent = element.aboveMe(); // Returns the parent element
    parent = element.aboveMe('.myClass'); // Returns the first parent element having this class

    // Object.keys(Function(Key, Value)) -> executes Object.keys(object) method,
    // returns all the keys... If a function was passed it loop the keys providing
    // the key and value of the property

    var keys = myObject.keys(function(key, value){
        ...
    });
