Bindings

One Way

We can achieve One Way Data Binding simply by defining the delimiter in the space or attribute we need the value to be.

If we don't want to do it this way, exposing the delimiter in the actual element attribute, we can still use the command e-attribute="{{ expression }}", it creates the attribute after being compiled. This commands is very versatile, we will go into details about it in the commands section.

Two Way

We can achieve Two Way Data Binding using the command e-bind:attribute="scopeProperty", by default it binds value attribute, it means that if we want to bind the `value` attribute of an input we can define like this e-bind="scopeProperty".
