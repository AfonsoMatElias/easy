What is Easy.js?

Easy.js, short for Easy and Asyncronous Javascript, is a javascript library for building user interfaces, and helps in the web applications development, providing a synchronous interaction between user interfaces and Javascript data.

It is a very easy library to get the hang of, with a simple command so you can see the results on the screen, it is totally focused on front-end development, it can be used to create the entire application as in specific parts of it. It is also good at building Single Page Applications, as it has the concept of web components.

Easy does not use the Virtual DOM concept to update data in the UI, for this reason it associates each property of the current instance with the fields where it was requested in the UI, and whenever a change occurs in property, it updates all the fields associated with it.
