Sem Virtual DOM

O Easy escuta cada propriedade (disponibilizada pela instância) na aplicação, quando a propriedade muda, ele atualiza o elemento.

Reutilize elementos

O Easy dá a você a oportunidade de reutilizar elementos já definidos na aplicação, como se fosse um copiar e colar.

O que é Easy.js? É uma biblioteca javascript para construção de interfaces de usuário, e auxilia no desenvolvimento de aplicações web, proporcionando uma interação síncrona entre interfaces de usuário e dados Javascript. Em algumas formas e comandos, é muito semelhante ao Vue.js e Angular.js, mas lida com as coisas de forma diferente!
