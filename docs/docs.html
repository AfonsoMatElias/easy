Introduction

What is Easy.js?

Easy.js, short for Easy and Asyncronous Javascript, is a javascript library for building user interfaces and that helps in the development of web applications, providing a synchronous interaction between user interfaces and Javascript data.

So, Why Easy.js?

As the name says, it is a javascript library really easy to use that provides to you a simple way to interact with your HTML by using Reactive Properties, Scoping data in the HTML, reusable components written in pure HTML, api calls using only attributes, and so on... It's very good and simple to build Single Page Application. No Virtual DOM is used, it uses the Real DOM to handle all the data changes.

Browsers Compatibility

It was tested in almost every popular browsers and IE 11, below that is not suported, because it uses some comcepts of ECMAScript 2015 (ES6) .

Scale of Projects

A web application fully developed with easy.js was tested only with small projects, larger projects was not tested yet.

Installation

It is easily added to a web project, just like the other libraries and framework, and for that there are two ways.

Locally

To use it locally, you need to download the file and import it

Download 


    <script src="/js/easy.js"></script>

Or, you can use...

CDN

Using CDN is a little bit simpler because the only thing that needs to be done is to import it 


    <script src="https://cdn.jsdeliv.com/gh/afonsomatelias/2.0.0/easy.js"></script>

Instance

When you import Easy it will added a function to the global object, so, to begin using it you need to create a new instance of Easy Class: 


    var app = new Easy(rootElement, options);
